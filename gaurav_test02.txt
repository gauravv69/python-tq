Q1. Write key points about each collection in python (12 Marks)
-> The following are the collections in Python:
   List: They are mutable 
	 ordered
	 allows duplicates.	
    	 eg: [1, 2, 3]
   Tuple: They are immutable 
	  ordered 
	  allow duplicates.
   	  eg: (1, 2, 3)
   Set: Set's are mutable
	They're unordered
	They allow no duplicates 
   	eg: {1, 2, 3}
   Dictionary: Dictionary consists key-value pairs
	       They are unordered 
	       The keys are unique in dictionary
               Eg: {"a:1", "b:2"}

Q3. Write a code to accept the number & check number is Armstrong or not (5 Marks)
-> num = int(input("Enter a number: "))
   og = num
   result = 0

   while num > 0:
	digit = num % 10
	result += digit ** 3
	num = num // 10

 if result == og:
	print(og,"is an Armstrong number")
else:
	print(og,"not an Armstrong number")
